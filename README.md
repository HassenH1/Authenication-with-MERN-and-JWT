## Build With

- React.js
- Express
- Node
- MongoDB
- Bootstrap

# Getting Started

## Installation

```bash
npm i
```

## Usage

### Start Server

```bash
cd server && nodemon server.js
```

### Start React app

```bash
npm run start
```

### Run Test

```bash
npm run test
```
